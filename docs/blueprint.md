# **App Name**: Arena Ace

## Core Features:

- Admin Authentication: Secure login for administrators to manage the event.
- Team Management: Dynamic admin interface to add, update, and manage teams, including team name, group assignment, captain name, and captain UID. Real-time updates reflected on public display.
- Match Management: Dynamic admin interface to create, schedule, and manage matches, specifying day, match, and auto-filling teams. Includes features to lock matches and prevent score changes.
- Score Entry: Dynamic admin interface to enter scores per team, including kills and placement, with automatic point calculation. Real-time validation and error handling to ensure data accuracy.
- Leaderboard Engine: Automated leaderboard calculation and display based on score entries, with sorting rules for each day. Dynamic updates with animations to highlight changes in rankings (e.g., teams moving up or down).
- Qualification Engine: Automatic selection of qualified teams based on leaderboard standings, with admin confirmation. Dynamic display of qualified teams with visual cues (e.g., highlighted names, special badges).
- Public Display Pages: Publicly accessible pages to view leaderboards and bracket information, updated live with smooth animations and transitions. Includes features to highlight qualifiers and grey-out eliminated teams.
- Dynamic Updates and Announcements: Admin interface to post real-time updates, announcements, and notifications to the public display pages. Includes features for scheduling announcements and targeting specific audiences.

## Style Guidelines:

- Primary color: Fiery Red (#FF4646), reflecting the intensity of Free Fire battles.
- Background color: Dark Grey (#333333) for a sleek, modern look that reduces eye strain.
- Accent color: Electric Blue (#29ABE2) to highlight key information and interactive elements.
- Body and headline font: 'Bebas Neue', a condensed sans-serif font, for a bold, energetic aesthetic.
- Use sharp, vector-based icons to represent in-game items and actions, maintaining a consistent style with the game's theme.
- A dynamic, grid-based layout that adapts to various screen sizes and devices, ensuring optimal viewing on any platform.
- Use fast-paced animations to transition between pages and highlight changes in the leaderboard, mirroring the game's high-octane action. Implement animations for team movements (up/down) in the leaderboard.
